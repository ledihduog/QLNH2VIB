Bước 3: Migration Model to DB
Vì dự án code theo kiểu code first chứ k phải DB first nên cần migration để ánh xạ model tạo bảng và DB từ code
Chạy lệnh: PM> dotnet ef migrations add QLNH20250821 --context QLNHContext
sau đó chạy lệnh: PM> dotnet ef database update --context QLNHContext
Lưu ý cần cd vào folder chứa project và cài Microsoft.EntityFrameworkCore.Tools mới chạy được lệnh tại PM>
